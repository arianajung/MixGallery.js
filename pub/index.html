MixGallery.js

Simple and lightweight Javascript library

where you can display all your multimedia in one compact gallery.

Demo

Display your multimedia in one simple gallery.

Photo, audio, audio, link - just about everything you need.

Features

Display various multimedia

Mix up the elements you wish to display ranging from photo, video, audio, and link. You are not limited to just one type!

Title and Caption

Simply click on each media element to enter an enlarged view of the media and see their title and caption. Make the caption as lengthy as you want!

Filter

Filter the elements in the gallery by clicking on the type of media at the top of the gallery.

Navigation

In the enlarged view, click on the arrows to nagivate to the previous and next media element.

Link

View the iframe of a link element in the enlarged view. Click on the link icon beside the title to be directed to the full website.

Responsive Design

A responsive design for various browser sizes.

Get Started

Download

Download the javascript and css files from GitHub.

Initialization

Link the .css file into the <head> section of your HTML document:

<link rel="stylesheet" type="text/css" href="MixGallery.css"></link>

Add the .js file into the <body> section of your HTML document:

<script type="text/javascript" src="js/MixGallery.js"></script>

Create an empty <div> element in the DOM and specify its id:

<div id="gallery"></div>

In your .js file, create an array of objects where each object represents each media element in the gallery.

const elements = [
    {
        src: "./image.jpg",
        type: "img",
        title: "A title.",
        caption: "This is an example of a caption.",
        priority: 2,
    },
    {
        src: "./video.mp4",
        type: "video",
        title: "Another title.",
        caption: "Another caption!",
        priority: 1,
    }
]

Lastly, create a new instance of MixGallery and pass in the <div> that was created earlier. Call render() to invoke MixGallery.

const mixgallery = MixGallery(document.querySelector("#gallery"), elements);
mixgallery.render();

API

Object Specificiation

An object representing a single media element should be specified as the following:

{
    src: "./image.jpg", // image source
    type: "img", // type of the media element
    title: "A title.", // title
    caption: "This is an example of a caption.", // caption
    priority: 2, // importance level of this media element
}

Description of each attribute:
  • src: source of the element
  • type: type of the element; choose from photo, video, audio, or link
  • title: title for the element
  • caption: caption for the element
  • priority: value of 1, 2, or 3 indicating the importance of this element (specifies size for the element where 1 gives the smallest size and 3 the biggest size)


Public Methods

A few simple methods can be called on a MixGallery instance:

const mixgallery = MixGallery( ... );

.render()

Triggers the initialization of MixGallery instance.

mixgallery.render()

.destroy()

Destroys the entire MixGallery instance.

mixgallery.destroy()
